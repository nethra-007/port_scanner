# 🔍 Python Port Scanner

A simple Python-based port scanner that scans open ports on a target system.

## 🚀 Features
- Scans ports from 1 to 1024
- Identifies open ports
- Helps understand basic networking concepts

## 🛠️ Technologies Used
- Python
- Socket module

## ▶️ How to Run
1. Open terminal
2. Navigate to project folder
3. Run the script:

python port_scanner.py

4. Enter target IP (example: 127.0.0.1)

## 📌 Output
Displays open ports of the given system.

## 📚 Learning Outcome
- Understanding of port scanning
- Basics of networking
- Python socket programming
